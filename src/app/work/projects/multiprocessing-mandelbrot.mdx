# Parallel Mandelbrot Renderer

This project explored parallel computing through fractal generation. I built a C-based renderer that used both multiprocessing and multithreading to accelerate a sequence of Mandelbrot frames and generate a zooming animation more efficiently than a single-threaded approach.

Repository: [View on GitHub](https://github.com/jesse-rost/my-portfolio/tree/main/multiprocessing-mandelbrot)

## What I built

- A multiprocessing framework using `fork()` and `exec()`
- Multithreaded rendering for independent pixel workloads
- Configurable process/thread settings for performance testing
- A frame-based workflow that produced a zoom animation

## Why it mattered

The project was a practical example of embarrassingly parallel work: each pixel calculation is independent, so concurrency can deliver major speedups. It also showed how performance depends on balancing process overhead, thread usage, and available hardware resources.
