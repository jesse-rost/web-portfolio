# Single-Cycle ARM Processor & SoC

This project was a full hardware-software build: I designed a 32-bit single-cycle CPU in VHDL, integrated it into an FPGA-based SoC, and ran bare-metal firmware on a physical robot platform. The system handled instruction decoding, register-file access, ALU operations, memory-mapped I/O, and autonomous navigation logic without relying on a soft-core processor.

Repository: [View on GitHub](https://github.com/jesse-rost/my-portfolio/tree/main/arm-single-cycle-processor)

## What I built

- A custom ARMv7-inspired single-cycle datapath in VHDL
- A hardwired control unit and ALU with flag handling
- Register-file, shifter, and memory-mapped peripheral logic
- Bare-metal firmware for robot control and sensor-driven navigation
- FPGA deployment on an Intel MAX 10 board driving a Digibot rover

## Why it mattered

The project was a strong introduction to computer architecture from the hardware side: I learned how instruction execution, control flow, and peripheral interfacing come together in a real system. It also reinforced how low-level design decisions affect timing, correctness, and physical behavior.
